# 智慧社区监控大屏 — 项目使用文档（用户手册）

## 一、系统简介

智慧社区监控大屏管理系统是一款面向物业公司、社区管理人员、安保人员的综合性管理平台。系统整合了社区概况、实时监控、设备管理、告警处理、人员信息、房屋信息、车辆信息、巡更管理、报修管理、公告信息等十大功能模块，提供数据可视化展示与业务管理的一体化解决方案。

**适用角色**：
- 物业管理人员
- 社区安保人员
- 系统管理员
- 住户/租户（查看公告、提交报修）

## 三、通用操作说明

### 3.1 登录系统

系统默认进入演示模式，首次访问自动设置默认用户，无需手动登录。

如需启用真实登录：
1. 访问 `/login` 页面
2. 输入用户名和密码
3. 点击登录按钮

> **注意**：当前为演示环境，路由守卫已配置为自动跳过登录。生产环境需修改 `src/router/index.js` 取消注释 `next('/login')`。

### 3.2 页面导航

- **侧边栏导航**：点击左侧菜单切换不同功能模块
- **面包屑导航**：页面顶部显示当前位置，可点击返回上级
- **页面切换**：所有页面切换均为单页应用无刷新跳转

### 3.3 搜索功能

各数据列表页面上方均提供搜索框：

1. 在搜索框输入关键词（如设备名称、人员姓名、房号等）
2. 按回车键或点击搜索按钮
3. 表格数据将实时过滤显示匹配结果
4. 点击重置按钮清空搜索条件

### 3.4 筛选功能

数据表格上方提供筛选条件：

1. 点击筛选下拉框（如状态、类型、级别等）
2. 选择需要的筛选值
3. 表格数据将自动过滤
4. 支持多个条件组合筛选
5. 点击重置按钮清空所有筛选条件

### 3.5 分页功能

数据表格底部提供分页控件：

- **页码切换**：点击页码或上一页/下一页按钮
- **页大小调整**：下拉选择每页显示 10/20/50/100 条
- **总条数显示**：右下角显示当前页/总条数

### 3.6 数据导出

各数据表格均支持导出 Excel：

1. 点击表格右上角的「导出 Excel」按钮
2. 系统自动将当前筛选后的数据导出为 `.xlsx` 文件
3. 导出文件包含表格当前显示的所有列

> **提示**：当前版本导出前端当前页数据。如需导出全部数据，请先调整页大小为最大值。

### 3.7 实时刷新

- **社区概况页**：数据每 30 秒自动刷新一次，无需手动操作
- **其他页面**：进入页面时自动加载最新数据，切换页面时刷新

### 3.8 全屏模式

- **实时监控页**：双击任意监控画面可进入该画面全屏模式；按 `Esc` 或再次双击退出全屏
- **全局全屏**：点击顶部栏的全屏按钮可将整个浏览器窗口切换为全屏

## 四、各模块详细说明

### 4.1 社区概况

**入口**：侧边栏「社区概况」

**功能说明**：
社区概况是系统的首页，展示社区运行的核心指标与趋势。

**页面内容**：
- **统计卡片**：
  - 常住人口数
  - 总户数
  - 车位使用率
  - 设备在线率
- **趋势图表**：
  - 人口趋势（近 12 个月）
  - 户数趋势（近 12 个月）
- **监控概览**：
  - 设备状态概览
  - 访客统计
  - 告警统计
  - 车辆进出统计

**操作说明**：
- 数据每 30 秒自动刷新
- 趋势图表支持鼠标悬停查看具体数值

---

### 4.2 实时监控

**入口**：侧边栏「实时监控」

**功能说明**：
展示社区各监控点位的实时画面，支持多画面布局切换。

**页面内容**：
- **布局切换按钮**：4 画面 / 6 画面 / 9 画面
- **监控点位卡片**：点位名称、在线状态、视频区域

**操作说明**：
1. 点击「4画面」「6画面」「9画面」按钮切换布局
2. 双击任意视频区域进入该画面全屏
3. 全屏模式下按 `Esc` 或再次双击退出
4. 离线点位显示离线提示画面

---

### 4.3 设备管理

**入口**：侧边栏「设备管理」

**功能说明**：
管理社区内所有弱电/智能化设备台账，包括门禁、摄像头、道闸、电梯对讲等。

**页面内容**：
- **统计卡片**：总设备数、在线数、离线数、故障数
- **图表**：设备状态分布饼图、设备类型分布柱状图
- **设备表格**：设备编号、名称、类型、位置、状态、IP 地址、最后在线时间

**操作说明**：
1. **搜索**：输入设备名称或编号快速查找
2. **筛选**：按设备类型（摄像头/门禁/道闸/电梯对讲/广播/传感器）或状态（在线/离线/故障）筛选
3. **查看详情**：点击表格行「查看」按钮，弹出设备详细信息弹窗
4. **导出**：点击「导出 Excel」导出设备台账

### 4.4 告警管理

**入口**：侧边栏「告警管理」

**功能说明**：
集中处理社区内所有设备告警与安防告警，支持告警确认与状态追踪。

**页面内容**：
- **统计卡片**：今日告警数、待处理数、已处理数、告警级别分布
- **图表**：告警趋势堆叠图、告警类型分布饼图
- **告警表格**：告警时间、类型、级别、位置、状态、描述

**操作说明**：
1. **搜索**：输入告警位置或描述关键词
2. **筛选**：按告警级别（紧急/重要/一般）或状态（待处理/已处理）筛选
3. **标记处理**：点击「处理」按钮，将告警状态更新为已处理
4. **导出**：点击「导出 Excel」导出告警记录

> **注意**：紧急级别告警以红色高亮显示，请优先处理。

## 六、常见问题

### Q1：为什么页面数据看起来是模拟的？

当前系统运行在**演示模式**，所有数据由 MockJS 动态生成，用于功能展示和界面效果验证。接入真实后端后，数据将来自真实数据库。

### Q2：如何切换到真实数据？

1. 后端开发完成后，确保接口路径与 Mock 定义一致（前缀 `/api`）
2. 修改 `src/main.js`，注释掉或删除 `setupMock()` 调用
3. 配置环境变量 `VITE_API_BASE_URL` 指向真实后端地址
4. 重新启动项目

### Q3：导出 Excel 的文件保存在哪里？

点击「导出 Excel」后，文件会自动下载到浏览器的默认下载目录（通常是「下载」文件夹），文件名格式为 `[页面名称]_导出_YYYYMMDD_HHmmss.xlsx`。

### Q4：实时监控画面为什么是静态的？

演示环境使用静态图片模拟视频流。接入真实监控系统后，将视频流地址（如 HLS/FLV/WebRTC）配置到监控点位数据中即可播放真实画面。

### Q5：如何修改自动刷新间隔？

社区概况页的自动刷新间隔在 `src/views/Dashboard/index.vue` 中配置：

```js
refreshTimer = setInterval(fetchData, 30000) // 30000ms = 30秒
```

修改该数值即可调整刷新频率。

### Q6：侧边栏折叠后如何恢复？

点击侧边栏底部的折叠按钮（或在窄屏下点击顶部菜单按钮）即可展开。折叠状态会自动记忆在当前会话中。

### Q7：系统支持手机访问吗？

系统已做基础响应式适配，侧边栏在窄屏下会自动折叠，表格支持横向滚动。但主要设计目标为桌面端大屏展示，移动端体验可能受限。如需专用移动端，建议另行开发 H5 版本。

### Q8：如何添加新的功能页面？

1. 在 `src/views/` 下创建新页面目录和 `index.vue`
2. 在 `src/router/index.js` 中添加路由配置
3. 在 `src/api/modules/` 下添加对应 API 接口
4. 在 `src/mock/modules/` 下添加 Mock 数据（开发环境需要）
